# Overview

The AI-Powered Developer Portfolio is a modern frontend application designed to present technical expertise, software engineering projects, certifications, and career achievements through an interactive and responsive interface.

Unlike traditional portfolio websites, this project integrates a locally hosted AI assistant powered by Ollama, allowing visitors to interact using natural language while exploring portfolio content.

The project emphasizes clean architecture, responsive design, performance optimization, security best practices, and maintainable code organization.

# Installation

Clone the repository.

```bash
git clone https://github.com/vivanpandya/portfolio.git
```

Move into the project directory.

```bash
cd Portfolio
```

Run the local development server.

```bash
python run.py
```

Open the application.

```
http://localhost:8000
```

---

# AI Assistant Setup

Install Ollama.

Verify installation.

```bash
ollama --version
```

Download the required language model.

```bash
ollama pull llama3
```

Start Ollama.

```bash
ollama serve
```

Run the model.

```bash
ollama run llama3
```

The AI Portfolio Assistant is now ready for use.
